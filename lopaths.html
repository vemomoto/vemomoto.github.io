Installation¶

The package can be installed via pip. To install the package, you can use

pip install lopaths

Please note that a compiler may be needed to install the package. Please refer to the section Installation on the main page.

Note

Some algorithms implemented in this package rely on share memory libraries that work on Unix systems only. If locally optimal paths shall be computed in large road networks, it is strongly encouraged to execute the code on Linux, as some tasks are not implemented to run in parallel on Windows.

Scientific Publication¶

The algorithms implemented in this package are explained in the paper “Locally optimal routes for route choice sets” (preprint). Please cite this publication if you have used the package in your own research.
