Installation¶

The package can soon be installed via pip. To install the package, you can then use

pip install hybrid_vector_model

Scientific Publication¶

The theory behind the model implemented in this package is explained in the paper “A hybrid gravity and route choice model to assess vector traffic in large-scale road networks” (preprint). Please cite this publication if you have used the package in your own research.
